## Bilder hochladen

1. Im CMS auf **Medien** klicken
2. **Upload** Button klicken oder Dateien per Drag & Drop ablegen
3. Unterstuetzte Formate: JPG, PNG, WebP, GIF, SVG, PDF

## Bilder verwenden

Hochgeladene Bilder koennen in Sections und Collections verwendet werden.
Die Media Library zeigt alle Bilder als Grid oder Liste.

## Bild-Details

Fuer jedes Bild kann bearbeitet werden:
- **Alt-Text** - Wichtig fuer SEO und Barrierefreiheit
- **Breite / Hoehe** - Wird automatisch erkannt

## Verwendungs-Check

Die Media Library zeigt an, wo ein Bild verwendet wird (in welchen Seiten oder Collections).
So kann man sicher sein, dass ein Bild nicht mehr gebraucht wird, bevor man es loescht.
